#### Status Indicators

.Function Indicators
[options="autowidth",options="header"]
|====================
| Character | Purpose
| `A` | Text output aborted 
| `G` | Garbage collection occurring
| `L` | Lt. Kernal port lock
| `P` | Text output paused
| `S` | Swapping data
|====================

.File transfer indicators
[options="autowidth",options="header"]
|====================
| Character | Purpose
| `F` | RS232 flow control asserted 
| `I` | Interrupts disabled
| `L` | Loading data?
| `O` | Overrun
|====================
